# student-performance-ml
# Student Performance Prediction using Machine Learning

## Overview
This project applies supervised machine learning techniques to predict student academic performance based on input features.

## Dataset
A publicly available student performance dataset was used for educational purposes.

## Methodology
- Data loading and inspection
- Data preprocessing
- Model training using classification algorithms
- Model evaluation

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

## Results
The models were evaluated using accuracy-based metrics. The project provided practical understanding of machine learning workflows.
